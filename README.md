## Introduction

A application to show different animation we can do in react native.

---

<br/>
<br/>

## :sparkles: Animations

- Horizontal Parallax Effect

- Vertical Parallax Effect

- Carousel Zoom Effect

- Background Change Blur Effect

- Slow Loading Effect

- Animated Onboarding

- Onboarding
